### Minor Changes

- Initial public release of refpool: a keyed, reference-counted, bounded LRU
  resource pool (`@refpool/core`), Prometheus/OpenTelemetry metrics, database
  adapters for node-postgres, TypeORM, Drizzle, Prisma, and Knex, and a NestJS
  integration.
